# AI Interview Coach with ATS Analyzer 

AI-powered Interview Preparation Platform built with MERN Stack.

## Features

- ATS Resume Analyzer
- Resume vs Job Description Match
- AI Mock Interviews
- Live Coding Interview Platform
- Real-time Code Compilation
- Camera & Voice Recording
- Authentication System
- Role-based Interview Questions
- Coding Test Cases
- Mobile Responsive UI

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Shadcn UI

Backend:
- Node.js
- Express.js
- MongoDB

Other Features:
- Docker
- Piston Compiler API
- JWT Authentication

## Roles Supported

- MERN Developer
- Java Developer
- Data Analyst
- Full Stack Developer
- Software Tester
- Software Engineer
- HR Round

## Installation

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```
